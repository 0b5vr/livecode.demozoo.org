What is Demoscene ?

The demoscene is an international computer art subculture focused on producing demos: self-contained, sometimes extremely small, computer programs that produce audiovisual presentations.
Some references to learn more about the demoscene:
Wikipedia article
Teach Yourself Demoscene in 14 Days

What is Live Coding ?

Live Coding is a contributive performance with one or multiple artist using code to generate real-time graphics and / or music.

What is Shader Showdown

2 performers, 25min, same starting screen, sit down and code from memory something that will make the audience vote for you.
Usually done using a tool called Bonzomatic

What is Shader Jam

Multiple live coders just jamming some visuals for a music act / DJ set.

What is Shader Royale

Bigger number of live coders (not just 2 from shader showdown), starting at same time, getting eliminated by the audience votes in a battle royale format.

What is Byte Battle

Similar to Shader Showdown, but using the fantasy console TIC80, and limited to 256 bytes is code length.
